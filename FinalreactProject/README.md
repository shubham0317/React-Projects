# React Collection Explorer

A modern React application that allows users to search and explore collections using an interactive and clean UI. The project demonstrates practical usage of **React Router, Redux, reusable components, and API integration**.

## 🚀 Features

* 🔍 Search functionality for finding collections
* 📂 Collection listing and detail display
* 🧭 Navigation using React Router
* ⚛️ Global state management with Redux
* 🔔 Toast notifications using React Toastify
* 🎨 Responsive UI with reusable components
* 📑 Clean project structure following component-based architecture

## 🛠 Tech Stack

* **React.js**
* **Redux**
* **React Router DOM**
* **React Toastify**
* **JavaScript (ES6+)**
* **CSS**
